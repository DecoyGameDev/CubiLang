# Hello World Example (yikes)

`
lib{
    output.cubl
}

global{
    READ(v)
}

hidden{
    class{
        define{
            'Helloworld.class'
        }
        data{
            'Hello World!'
        }
        cub=if{
            bool, On == IStrue{
                prntout(data^^)
            }
        }
    }
}

numb{
    define{
        'On', 'Off'
    }
    int 1
    int 0
    {define bool}
}`
